# TodoList
## Что это?
Приложение task tracker с тремя досками(todo, in progress, done).
## Функционал
+ Добавление задач
+ Изменение задач
+ Фильтрация задач по наличию комментариев, описания или тегом
## Стек
+ JS
+ Redux
+ React
+ React Router
+ React Formik
+ CSS
